Create your next color palette

Click “Generate” to get five fresh colors. Tap a swatch to copy its HEX.

How it works

  1. Generate : Press the big button (or hit the spacebar) for five random colors.
  2. Copy : Click any swatch; its HEX code is copied to your clipboard and confirmed by a toast.
  3. Tweak : Keep generating until the palette feels right—perfect for quick mock‑ups or style guides.

Tech stack

  • Semantic HTML5
  • Vanilla CSS + flexbox (no framework, but ready for Tailwind/Bootstrap drop‑in)
  • Tiny JavaScript module for random color creation, copy‑to‑clipboard, smooth scrolling, and mobile nav toggle
